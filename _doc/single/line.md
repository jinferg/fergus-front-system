## Line

The Single line `s-line` creates a soft horizontal rules that divide sections.

<span class="s-line"></span>

```html
<span class="s-line"></span>
```

### Options

The Single Line has 2 local variables.

```css
@s-line-width: 1px;
@s-line-color: @shade-light;
```
